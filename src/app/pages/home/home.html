¿Qué hace único a La Codicia del Sabio?

⚜️
⚡

Sistema de Recursos Dual

Energía para guerreros y Maná para hechiceros. Dos caminos de poder completamente diferenciados.

🟢

3 Acciones por Turno

Cada turno tienes 3 acciones para atacar, moverte o usar habilidades. Modificable por condiciones como Acelerado o Ralentizado.

🛡️

Escudo + Vida

Sistema de doble capa de protección. El escudo absorbe daño antes que la vida, añadiendo profundidad táctica.

🎯

Especialización o Multiclaseo

Domina una clase hasta el nivel 12 con Pasiva de Maestría, o combina varias clases para crear builds únicos.

📱

App Compañera

Gestiona personajes, campañas y combates con nuestra app "La Codicia del Sabio", disponible para móvil.

🎲

Mecánicas Intuitivas

Mecánicas originales y sencillas de entender: DA, margen de daño, condiciones únicas y sistema de enemigos por tiers.
